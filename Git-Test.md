# Работа по работе с Git

## Основыне команды Git
**git add - добавить версионность**

**git int - инициализируем**

**git commit - по сути сохранение текущей сборки в репозитории** 

**git log - журнал измененний**

**git checkout - перейти на ветку**

**git diff - узнать разницу между сохранненым и текущим состоянием**
